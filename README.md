# Optimizing Doctor Availability and Appointment System

## 📌 Overview
This project is a healthcare management system designed to optimize doctor availability and streamline appointment scheduling using digital technology and AI concepts.

## 🚀 Features
- Patient, Doctor, and Hospital modules
- Appointment booking and scheduling system
- Real-time appointment status tracking
- Doctor availability management
- Email notification system (conceptual)

## 🧠 Key Highlights
- Focused on reducing patient waiting time
- Optimized doctor scheduling using intelligent logic
- Designed scalable system architecture

## 🛠️ Technologies Used
- Frontend: React
- Backend: Spring Boot
- Database: MySQL
